matevfs.FileInfo

matevfs.FileInfo — Object which stores information about a file

Synopsis

class matevfs.FileInfo:

Attributes

atimeAccess time.
block_countSize measured in 512-byte blocks
ctimeChange time.
deviceDevice type. This property is only valid if the file is local.
flagsFlags for this file. See matevfs File Flags.
gidGroup ID of the file
inodeInode number. This property is only valid if the file is local.
io_block_sizeOptimal buffer size for reading/writing the file.
link_countNumber of links to this file.
mime_typeMIME type as an ASCII string
mtimeModification time.
nameBase name of the file (no path).
permissionsFile permissions.
sizeSize in bytes.
symlink_nameIf the file is a symlink (see `flags'), the name of the file this link points to.
typeFile type (i.e. regular, directory, block device...).
uidUser ID of the file
valid_fieldsFields which are actually valid in this structure.
